Mulai dari hari pertama masuk mengenai pengenalan mata kuliah PPM, sejarah teknologi dan handphone, kemudian mempelajari Github beserta command - command Git dan GUI nya , lalu masuk ke dalam JQuery Mobile beserta fitur - fitur apa saja yang bisa dibuat menggunakan JQuery Mobile, hingga sampai sejauh ini sampai belajar mengenai AngularJS.

Untuk pembelajaran mengenai Github, Github adalah salah satu tempat dimana kita bisa mengupload project dan menyimpan project kita di server yang telah disediakan. Dari Project yang sudah kita upload ini ada banyak hal yang bisa dilakukan terutama untuk bisa diupdate oleh kita maupun orang luar (semacam kita berkolaborasi untuk bisa membuat program kita menjadi lebih baik lagi).

Mengenai penggunaan Github beserta fitur - fiturnya, kita perlu belajar banyak command yang nantinya kita masukkan melalui GitBash yang diinstal ke komputer kita (bila tidak mau melalui command CLI, bisa juga menggunakan GUI yang tersedia dari berbagai pihak). Berikut adalah beberapa command dasar yang ada :

  • Git Init : Perintah untuk membuat repository local di komputer kita (umumnya di folder yang kita pilih). Dari repository kita ini nanti bisa melakukan banyak hal
  • Git Add : Memasukkan suatu file ke dalam staging area. Staging area adalah suatu keadaan dimana file dipersiapkan untuk di commit. Selama di staging area , file bisa saja di edit (diganti dengan versi staged yang lain, dikeluarkan dari staged area , dll)
  • Git Commit -m "message" : Untuk mengcommit / mengkonfirmasi suatu perubahan yang akan dimasukkan ke dalam repository. Biasanya disertai dengan message yang memberi tahu ini commit untuk melakukan apa.
  • Git Remote add origin https://github.com/morphlng7/TesPPM : Untuk menambahkan suatu repository bernama origin yang lokasi repositorynya berada di server github (dalam contoh ini menggunakan repository dari github saya sendiri)
  • Git Push origin master : Untuk mengepush / upload semua file yang ada di repository kita(master) ke repositroy lain (dalam hal ini origin yang ada di server github)
  • Git Pull origin master : Untuk mengepull / download semua file yang ada di repository di server ke local kita
